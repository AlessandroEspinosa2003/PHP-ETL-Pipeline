# PHP-ETL-Pipeline

This repository is as a personal documentation of work I've done in my Advanced Software Engineering course!

In the class, we have been tasked with creating an ETL Pipeline in PHP, using an NGINX webserver, MySQL database on a t3.micro AWS EC2 instance.
For the instance itself we are bottlenecked with 2 vCPUs, 30 GB of storage, and 1 GB of ram. This is a very small configuration!

This task is definietly a challenge and I want to keep track of what I am able to accomplish with this legacy configuration working on very limited hardware.
